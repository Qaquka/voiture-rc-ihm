Aide – Mise en route rapide

  1. Connexion réseau : connectez votre téléphone/PC au réseau de l’ESP32 (192.168.4.1 en AP ou votre LAN).
  2. Configurer les URLs : dans l’onglet Contrôles + Caméra, renseignez :
    • Flux MJPEG → ex. http://192.168.4.1:81/stream
    • Base HTTP → ex. http://192.168.4.1
  3. Tester la liaison : cliquez sur Test HTTP, puis vérifiez le statut (HTTP OK).

Commandes voiture

  • Endpoint: /cmd?c=A|R|G|D|S|C|0|1|2|3
  • ↑ / Z Avancer
  • ↓ / S Reculer
  • ← / Q Gauche
  • → / D Droite
  • Espace STOP moteur
  • Relâche direction Centre
  • 0 25%
  • 1 50%
  • 2 75%
  • 3 100%

Caméra & iPhone

  • Utilisez l’onglet Caméra seule pour un affichage plein écran.
  • Appuyez sur Plein écran puis Anti-veille si disponible.
  • En cas d’écran noir : Recharger puis vérifier l’URL MJPEG.
  • En paysage, le champ de vision est plus confortable.

Dépannage rapide

  • HTTP KO : vérifier la base HTTP et le Wi-Fi.
  • Flux indisponible : vérifier le port :81/stream.
  • Latence : réduire la distance Wi-Fi et fermer les applis en arrière-plan.
  • Sécurité : tester d’abord roues levées, garder le STOP accessible.

Télémétrie (optionnelle)

  • JSON: /telemetry (speed + battery)
  • Fallback: /speed et /battery
  • Le polling est HTTP uniquement (pas de WebSocket/MQTT).
