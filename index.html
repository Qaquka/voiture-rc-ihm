Aide – Mise en route rapide

  1. Connexion réseau : connectez votre téléphone ou votre PC au même réseau que l’ESP32.
  2. Configurer les URLs : dans l’onglet Contrôles + Caméra, renseignez :
    • Flux MJPEG → par exemple http://192.168.1.168:81/stream
    • Base HTTP → par exemple http://192.168.1.168
  3. Tester la liaison : cliquez sur Test HTTP pour vérifier que les commandes arrivent bien à l’ESP32.

Commandes voiture

  • Endpoint principal : /cmd?c=A|R|G|D|S|C|0|1|2|3
  • ↑ / Z : avancer
  • ↓ / S : reculer
  • ← / Q : tourner à gauche
  • → / D : tourner à droite
  • Espace : STOP
  • 0 1 2 3 : réglage de la puissance moteur

Logique de conduite

  • Il est possible de tourner et avancer en même temps.
  • Si on relâche avance/recul, le moteur s’arrête.
  • Si on relâche gauche/droite, les roues reviennent au centre.
  • Le bouton STOP arrête le moteur puis remet la direction au centre.

Caméra

  • L’onglet Caméra seule permet d’avoir uniquement le flux vidéo.
  • Le bouton Plein écran est pratique sur téléphone.
  • Le bouton Anti-veille peut éviter l’extinction automatique de l’écran selon le navigateur.
  • Si le flux se bloque, utilisez Recharger.

Dépannage / sécurité

  • Si les commandes ne répondent pas, vérifier l’adresse IP de base HTTP.
  • Si la vidéo ne s’affiche pas, vérifier le port :81/stream.
  • Faire les premiers essais avec les roues levées ou dans un espace dégagé.
  • Garder le bouton STOP accessible pendant les tests.
